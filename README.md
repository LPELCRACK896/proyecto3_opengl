# Proyecto 3: OpenGL

## Funcionalidades implementadas
- Musica con pygame
- Zoom in y zoom out
- Some Shadders
- 5 modelos cargados

## Controles
- WASD para moverse alrededor del objeto
- O y P para zoom
- IJKL para mover el pointlight en la escena
- F1, F2, F3, F4, F5 para cambiar modelos. 
- Z, X, C, V, B, N para alternar shadders
